API Spec
========

`POST` `/api/users/` - create a new user

Sample request:

    {
      "username": "tyler_durden",
      "password": "dQw4w9WgXcQ"
    }

Sample response:

    {
      "key": "9bZkp7q19f0.KmtzQCSh6xk"
    }

* * *
